# HTML & CSS Learning Project

This project is designed for students learning HTML and CSS.

## Prerequisites

Before running this project, you need to install the following tools on your computer:

### 1. Visual Studio Code

Visual Studio Code is a free code editor that makes it easy to write and edit HTML and CSS.

**Installation:**
- Visit [https://code.visualstudio.com/](https://code.visualstudio.com/)
- Download Visual Studio Code for your operating system
- Follow the installation instructions for your OS (Windows, macOS, or Linux)
- Once installed, you can open this project folder in VS Code to start coding

### 2. Node.js (LTS version)

Node.js is required to run the local development server.

**Installation:**
- Visit [https://nodejs.org/en/download](https://nodejs.org/en/download)
- Download the LTS (Long Term Support) version for your operating system
- Follow the installation instructions for your OS (Windows, macOS, or Linux)
- Verify installation by opening a terminal/command prompt and running:
  ```bash
  node --version
  ```

### 3. Git

Git is a version control system that helps manage your code.

**Installation:**
- Visit [https://git-scm.com/install/](https://git-scm.com/install/)
- Download Git for your operating system
- Follow the installation instructions for your OS
- Verify installation by opening a terminal/command prompt and running:
  ```bash
  git --version
  ```

## Running the Project

Once you have Visual Studio Code, Node.js, and Git installed:

1. Open a terminal/command prompt
2. Navigate to the project directory
3. Run the following command:
   ```bash
   npx http-server
   ```

4. The server will start and display a URL (usually `http://localhost:8080` or `http://127.0.0.1:8080`)
5. Open your web browser and navigate to the displayed URL to view the project

## Stopping the Server

To stop the server, press `Ctrl + C` (or `Cmd + C` on macOS) in the terminal.
